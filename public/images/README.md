# Imagens do Produto

Coloque as imagens do produto Junta Nova nesta pasta com os seguintes nomes:

- **produto1.jpg** - Imagem principal do produto (pote único)
- **produto2.jpg** - Dois potes do produto
- **produto3.jpg** - Produto com destaque para joelhos
- **produto4.jpg** - Produto com pessoa idosa

## Dimensões Recomendadas

- Largura: 800px - 1200px
- Formato: JPG ou PNG
- Qualidade: Alta (para melhor apresentação)

## Onde encontrar as imagens

As imagens foram fornecidas no arquivo index.html original.
Copie-as para esta pasta com os nomes corretos.
